What is USeq?

USeq is a collection of 180+ Java applications for analyzing next generation sequencing data from ChIPSeq, RNASeq, BisSeq, DNASeq, Variant Annotation/ Filtering, Alignment/VCF QC, capture array design, IGV/ DAS2/IGB/UCSC file manipulation, etc. Both GUI and cmd line interfaces are available. USeq is under continuous development at the Huntsman Cancer Institute.

A description of some of the statistics and comparison of methods used in USeq is published, see Nix DA, Courdy SJ, Boucher KM: Empirical methods for controlling false positives and estimating confidence in chIP-seq peaks. BMC Bioinformatics. 2008 Dec 5;9(1):523. Note, USeq makes use of the DESeq2 R package to calculate p-values from a negative binomial distribution for differential enrichment/ reduction. See Love MI, Huber W, and Anders S (2014)

How to use/ install USeq?

To use USeq, download the latest release and install Java 1.7+. Some apps will require additional R libraries such as Storey's Q-Value and the Huber's lab DESeq2.
To launch an app such as 'ScanSeqs' type 'java -jar -Xmx4G yourPathTo/USeq/Apps/ScanSeqs' on the command line to print a menu and description of options.

Afraid of the command line interface? No problem, give the GUI_ClickMe.jar a try. Launch it on Mac's and Windows by double clicking the application. This runs a draggable/ point and click wrapper around the USeq applications.

Requirements

  • MacOSX or Linux, USeq apps are untested on Windows
  • Java 1.7+
  • Familiarity with command line programming

Common mistakes:

  • No java sourced or installed. Type 'java -version' on the command line. It should be 1.7+
  • Not using the -jar option. Type: 'java -jar -Xmx2G yourPathTo/USeq/Apps/ScanSeqs'
  • Out of memory error. Increase the heap size with the -Xmx flag. Type: 'java -jar -Xmx20G yourPathTo/USeq/Apps/ScanSeqs'. Keep the size < 0.9 of that installed.
  • Trying to run the apps on Windows. It is worth a try. These have been tested on Linux and Mac OSX.
